📊 Student Academic Performance Analysis (R)
A group data analysis project built in R, exploring how behavioural and socioeconomic factors influence university students' final grades.
Key highlights:

Dataset: 33-variable student prediction dataset covering demographics, family background, study habits, and academic records
Data wrangling: column renaming, NA removal, deduplication, and factor encoding
Analyses: correlation between note-taking frequency & grades, impact of parental education level, effect of study hours and class engagement
Visualisations: QQ plots, bubble charts, stacked bar charts, line charts using ggplot2
Hypothesis tested: whether consistent note-taking, active listening, higher maternal education, and 6–10 study hours per week predict a top grade (AA)

Tools: R, ggplot2, dplyr, tidyr, plotrix
